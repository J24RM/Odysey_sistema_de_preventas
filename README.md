# Odysey_Lab

- Ver todas las ramas que estan en remoto: git branch -r
- Clonar el repositorio "git clone link"
- Crear una nueva rama "git checkout -b (nombre de la rama)"
- Obtener las actaulizaciones hechas en el repositorio "git pull" (ANTES DE CREAR UNA LINEA PERSONAL, HACER UN PULL)
- Agregar cambios a la linea local "git add -A"
- Hacer commit en rama local "git commit -m "descripcion""
- Hace run push en rama local "git push"
- Pasos para unir ramas local con develop: 1."git checkout develop" 2."git pull" 3."git merge (nombre de la rama local)" 4."git push" 

Para tener una rama creada en remoda en local es con: git checkout (nombre de la rama)

Crear una nueva rama git checkout -b (nombre de la rama) y en momento de hacer push es: git push -u origin (nombre de la rama)
